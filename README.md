# ImageMatcher

**ImageMatcher** is a Java desktop application for searching and managing images. The app allows users to select a reference image and a folder, then find images within that folder that match based on color similarity, dimensions, timestamps, or visual similarity. It also supports cropping and saving images.

## Features

- Search images by color similarity  
- Find images with the same dimensions or size  
- Compare image timestamps  
- Detect visually similar images  
- Crop and save images efficiently  
- Intuitive and user-friendly interface
